# Module 3 Assignment

JavaScript Fundamentals Practice

## Topics Covered

- Variables
- Data Types
- Operators
- Conditionals
- Loops

## Files

| Task | File | Description |
|------|------|-------------|
| Task 1 | `student.js` | Store and display student information |
| Task 2 | `dataTypes.js` | Check data types using `typeof` |
| Task 3 | `calculator.js` | Basic arithmetic operations |
| Task 4 | `passOrFail.js` | Grade calculation using if-else |
| Task 5 | `forLoop.js` | Print numbers 1 to 10 |
| Task 6 | `whileLoop.js` | Print numbers 10 to 1 |

## How to Run in VS Code

1. Open this folder in VS Code: `File → Open Folder → module-3-assignment`
2. Open any `.js` file (e.g. `student.js`)
3. Press **F5** or go to **Run → Start Debugging** → choose **Run Current File**
4. Output will appear in the **Terminal** panel at the bottom

**Shortcut:** `Ctrl + Shift + B` runs the current file (Build Task).

## How to Run in Terminal

```bash
node student.js
node dataTypes.js
node calculator.js
node passOrFail.js
node forLoop.js
node whileLoop.js
```

Or use npm scripts:

```bash
npm run task1
npm run all
```
